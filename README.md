# 2023-TV-series-data-analysis
云顶书院大一研学收尾项目，是大一所学习内容的总结，爬取中国各个主流网站的电视剧数据进行数据分析，分析其中规律

#### 环境准备

- python 3.11
- pandas==2.0.3
- numpy==1.24.3
- matplotlib==3.7.1
- requests==2.28.1
- beautifulsoup4==4.11.1

#### 运行

1. 克隆代码到本地

   ```
   git clone https://github.com/YinHan-Zhang/Mutual-AI
   ```

2. 安装依赖

   ```
   pip install -r requriements.txt
   ```
